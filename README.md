# Pomodoro Focus - Odaklanma & Ambiyans Sesleri ⏱️

Pomodoro Focus, odaklanma sürelerinizi yönetirken arka planda dinlendirici sesler miksleyebileceğiniz modern bir Pomodoro aracıdır.

## 🛠️ Özellikler
- **Web Audio API Ses Sentezleyici:** Yağmur, esinti/orman, kafe uğultusu ve beyaz gürültü sesleri tarayıcı içinde gerçek zamanlı olarak matematiksel formüllerle sentezlenir. *Hiçbir dış ses dosyasına (MP3/WAV) bağımlı değildir!*
- **SVG Sayaç Halkası:** Zamanla uyumlu dolan dairesel görsel ilerleme göstergesi.
- **Seans İstatistikleri:** Tamamlanan seanslarınızı ve dakikalarınızı `localStorage` üzerinde kaydedip gün bazlı SVG barlar ile görselleştirme.
- **Ayarlar Paneli:** Çalışma ve mola sürelerini dilediğiniz gibi özelleştirme.
- **Light & Dark Tema:** Arayüz temasını anında değiştirebilme.

## 💻 Kullanılan Teknolojiler
- **Yapı:** HTML5 (SVG Dairesel Grafikler)
- **Tasarım:** CSS3 (Glassmorphism, CSS Animations, Dark/Light Mode)
- **Mantık:** JavaScript (Web Audio API, localStorage, ES6+)
- **İkonlar:** Lucide Icons

## 🚀 Nasıl Çalıştırılır?
Tarayıcıların ses çalma güvenlik politikalarından ötürü, bu projenin bir yerel HTTP sunucusu (örneğin Python HTTP sunucusu veya Live Server) üzerinden çalıştırılması önerilir.
